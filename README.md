# NeoVim用init.vimファイル
### 配置先
+ Windows  
C:\Users\[ユーザー名]\AppData\Local\nvim\init.vim  
※通常作成されないので自分で作る

+ Linux  
~/.config/nvim/init.vim  
※通常作成されないので自分で作る
